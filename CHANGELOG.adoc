= Auxly Changelog

== auxly-0.6.0 (2018-12-09)
=== Highlights
  - Updated CHANGELOG format.

=== Changed
  - Can now join paths via the `Path` and `File` constructors, e.g. `File("foo", "bar.txt")` is equivalent to `File(os.path.join("foo", "bar.txt"))`.

== auxly-0.5.5 (2018-10-28)
